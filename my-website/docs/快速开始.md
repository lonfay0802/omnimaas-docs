# 快速开始

OminiMaaS目前完全兼容原生OpenAI SDK、Claude SDK、Gemini SDK以及HTTPS协议，已经完成相关SDK接入的用户可直接通过更改BASE_URL
和API_key 来完成服务切换。只需几行代码，即可使用您偏好的 SDK 或框架快速上手。

可参考以下示例中的几个步骤完成服务对接，接口传参与响应可参照官方文档。

## 1. 平台登录
访问 [OmniMaaS](https://omnimaas.com) ，点击右上角“登录”按钮进行登录，还没账号的用户需要先进行注册后登录，目前仅支持手机号验证注册，注册完成后支持短信验证码、账户密码两种登录方式。

## 2. 选择模型
- 登录 OmniMaaS 控制台，在[模型广场](https://omnimaas.com)页面可查看平台当前已上架的模型名称、模型价格、模型详情、上下文长度等信息；
- 同时，用户可进入模型详情页查看模型的详细信息及其可用端点；

## 3. 获取API Key
- 登录 OmniMaaS 控制台，进入 [API Keys](https://omnimaas.com)页面。
- 新建密钥并妥善保存；每个请求都需携带 `Authorization: Bearer <API_KEY>` 头。
- 如需区分环境，可生成多个 Key 并设置标签或权限范围。

## 4. REST API调用

目前平台支持REST API进行调用，可通过各类Apifox、curl命令等方式进行测试；

```bash
curl https://api.omnimaas.com/v1/chat/completions \
  -H "Authorization: Bearer $OMNI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
        "model": "gpt-5.1",
        "messages": [
          {"role":"user","content":"解释下量子是什么？"}
        ]
      }'
```

## 5. OpenAI SDK调用
目前平台上架模型均支持OpenAI SDK库进行调用，可通过以下命令安装OpenAI Python库：

```bash
npm install openai
```

安装完成后，可参考以下Python代码完成OpenAI SDK库的使用和调用平台模型：

```python
from openai import OpenAI
client = OpenAI(api_key="OMNI_API_KEY", base_url="https://154.23.160.216:3000/v1/")

resp = client.chat.completions.create(
    model="gpt-5.1",
    messages=[
      {
        "role": "user",
        "content": "Hello from OmniMaaS!"
      }
    ],
)
print(resp.choices[0].message.content)
```
